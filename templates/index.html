How to Use

Quick steps to generate reports and manage TestRail content.

  • Reporter: pilih Project & Plan (klik Refresh bila daftar kosong), cari nama run di kolom search, centang run yang mau dilaporkan, lalu klik Preview Report. Report HTML terbuka di tab baru dan bisa langsung diunduh.
  • Management → Plan: isi nama plan dan (opsional) milestone, lalu submit. Form otomatis bersih setelah sukses.
  • Management → Run: pilih plan yang aktif, isi nama/desc/refs. Untuk scope: centang “Include all cases” atau klik Select cases untuk buka modal, cari/seleksi case (bisa “Select visible” atau “Clear”), lalu submit.
  • Management → Case: buat case baru dengan title + refs opsional. Jika ada BDD, tempelkan skenario di kolom BDD (satu baris per langkah).
  • Tips: tombol Refresh tersedia di plan/run/case picker. Lampiran akan dikompresi dan di-inline di report; video bisa ditranscode bila fitur video aktif.
