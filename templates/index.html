How to Use

Quick steps to generate reports and manage TestRail content.

Reporter

  • Pilih Project dan Plan dari dropdown (klik Refresh jika daftar kosong)
  • Gunakan kolom search untuk mencari run berdasarkan nama
  • Centang run yang ingin dilaporkan (bisa lebih dari satu)
  • Klik Preview Report - report HTML akan terbuka di tab baru dan bisa langsung diunduh
  • Tips: Lampiran akan dikompresi dan di-inline di report; video bisa ditranscode bila fitur aktif

Management - Hierarchical Navigation

Management menggunakan navigasi hierarki: Plans Runs Test Cases

Plans

  • Lihat daftar test plans (diurutkan berdasarkan update terbaru)
  • Setiap plan menampilkan: ID, status (Active/Completed), dan waktu terakhir diupdate
  • Gunakan search bar untuk filter plans berdasarkan nama
  • Klik tombol Refresh untuk mendapatkan data terbaru
  • Klik tombol Edit pada plan untuk melihat runs di dalam plan tersebut

Runs (dalam Plan)

  • Setelah klik Edit pada plan, modal akan muncul menampilkan daftar runs
  • Breadcrumb menunjukkan posisi: Plans > [Nama Plan]
  • Klik tombol Edit pada run untuk melihat detail run dan test cases
  • Klik tombol Delete untuk menghapus run (dengan konfirmasi)
  • Klik Back atau breadcrumb Plans untuk kembali ke daftar plans

Test Cases (dalam Run)

  • Setelah klik Edit pada run, modal akan muncul menampilkan detail run dan test cases
  • Breadcrumb menunjukkan posisi: Plans > [Nama Plan] > [Nama Run]
  • Edit Run Details: nama, deskripsi, dan references
  • Klik Save Changes untuk menyimpan perubahan run
  • Klik tombol Edit pada test case untuk mengedit case (title, refs, BDD scenarios, attachments)
  • Klik tombol Delete pada test case untuk menghapus (dengan konfirmasi)
  • Klik Back atau breadcrumb untuk navigasi ke level sebelumnya

Edit Test Case

  • Modal edit case akan muncul di atas modal run details
  • Edit Title (required), References, dan BDD Scenarios
  • Upload Attachments: drag & drop atau klik untuk browse (PNG, JPG, GIF, MP4, WebM, PDF, max 25MB)
  • Klik Save Changes untuk menyimpan
  • Klik Cancel untuk kembali ke run details tanpa menyimpan

Create New Content

  • Create Plan: Isi nama plan dan (opsional) milestone, lalu submit. Form otomatis bersih setelah sukses
  • Create Run: Pilih plan yang aktif, isi nama/desc/refs. Untuk scope: centang "Include all cases" atau klik Select cases untuk memilih case tertentu
  • Create Case: Buat case baru dengan title + refs opsional. Jika ada BDD, tempelkan skenario di kolom BDD (satu baris per langkah)
